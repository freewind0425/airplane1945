# Aircraft Bible

## Design Intent
기체는 스킨이 아니라 플레이 스타일이다. 모든 기체는 Build Starter 역할을 한다.

## Release Aircraft 6
| Code | Name | Role | Default Weapon |
|---|---|---|---|
| PK-A01 | Falcon | Balance / Assault | Straight Cannon |
| PK-B01 | Vulcan | Heavy / Tank | Scatter Cannon |
| PK-C01 | Hawk | Speed / Homing | Guided Missile |
| PK-D01 | Phantom | Energy / Laser | Laser |
| PK-E01 | Raven | Chain / Electric | Chain |
| PK-F01 | Titan | Destroyer / Bomb | Heavy Bomb |

## PK-A01 Falcon
- Project K의 대표 기체
- 레드 + 골드 + 화이트 + 블루 코어
- Variable Wing System
- Lv5에서 레일건, 에너지 윙, 4엔진 표현

## PK-B01 Vulcan
- 중장갑 강습 기체
- 블랙 + 골드 + 오렌지 코어
- 산탄/실드 빌드와 시너지
