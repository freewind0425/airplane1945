# Project K GDD v1.0.7

## Vision
Project K는 `멋진 기체를 성장시키며 계속 키우고 싶은 아케이드 슈팅 게임`이다.

## Game
GO AIR RAID

## Core Values
1. 기체가 멋있어야 한다.
2. 성장 체감이 강해야 한다.
3. 다시 접속할 이유가 있어야 한다.

## Current Direction
- 수동 조작 우선
- AUTO / OVER DRIVE는 추후 Phase에서 활성화
- Level Up은 즉시 팝업이 아니라 Level Up Core에 누적
- Boss는 5 Stage마다 등장
- 배속은 단일 버튼으로 x1 > x2 > x3 > x4 > x5 > x1 순환
